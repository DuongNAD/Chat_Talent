## 9. Technology Stack V2.0 — So Sánh Nâng Cấp

| Layer | V1.0 | V2.0 PRO | Lý Do Nâng Cấp |
|-------|------|----------|-----------------|
| **Hạ tầng** | Cài trực tiếp lên OS | **Docker Compose** | 1 lệnh start/stop, backup volume, migrate dễ |
| **Backend** | Express.js | **NestJS** | Module architecture, DI, Guards, chống spaghetti code |
| **Database** | PostgreSQL 16 + Prisma | PostgreSQL 16 + **pgvector** + Prisma | Vector search cùng DB, hybrid RAG SQL |
| **Vector Store** | SQLite-vss / File-based | **pgvector (hợp nhất)** | Không phân mảnh, 1 DB duy nhất |
| **AI Model** | gpt-4o | **gpt-4o-mini + Ollama fallback** | Giảm chi phí 90%, offline chat miễn phí |
| **AI Caching** | Không có | **Semantic Cache (pgvector + Redis)** | Câu hỏi tương tự → $0 API cost |
| **Video** | Download MP4 nguyên cục | **HLS Streaming (.m3u8)** | Tua đến đâu tải đến đó, mượt như YouTube |
| **FFmpeg** | CPU software | **Hardware Accel (NVENC/QSV)** | Không vắt kiệt CPU |
| **Upload** | Server-side processing | **Client-side Rust pre-processing** | Giảm 5-10x bandwidth upload |
| **UI Styling** | Vanilla CSS | **Tailwind CSS + shadcn/ui** | Dev nhanh, Dark Mode, Accessibility built-in |
| **Local Storage** | Không có | **tauri-plugin-sql (SQLite)** | Zero-latency UI, offline-first |
| **Distribution** | .zip qua Google Drive | **Tauri Auto-Updater** | Push update → app tự cài ngầm |
| **Auth** | Open registration | **Invite-Only codes** | Chống bot spam, kiểm soát truy cập |
| **Voice/Video** | Không có | **WebRTC P2P** | Voice chat, 0 bandwidth server |
| **Process Manager** | PM2 | **Docker restart: always** | Container-native, không cần PM2 |
| **Tunnel** | cloudflared service | **cloudflared container** | Cùng stack Docker, quản lý đồng nhất |
| **Ngôn ngữ UI** | Chưa quyết | **100% Tiếng Việt** | GenZ/thân thiện, gần gũi nhóm bạn |

## 11. Chốt Phương Án "Open Questions"

| Câu hỏi | Quyết định V2.0 | Chi tiết |
|----------|-----------------|----------|
| **Tên miền** | Mua `.xyz` hoặc `.net` trên Cloudflare Registrar | Rẻ nhất, auto-renew, DNS setup Tunnel siêu mượt |
| **CCU (người online)** | Stack cân 500-1000 CCU | Nhóm < 50 người → hệ thống "idle". Nút thắt: upload speed mạng nhà |
| **OpenAI Model** | `gpt-4o-mini` cho 100% | Function Calling thừa sức. **Hard Limit $5/tháng** trên Dashboard |
| **Rate Limit** | 30 câu Liva/user/ngày | Chống spam sạch tiền API |
| **Máy case specs** | i3/i5 Gen 8+, RAM 8GB, **SSD NVMe** | SSD bắt buộc cho DB I/O. Phân vùng riêng cho media |
| **Backend Framework** | **NestJS** | Module + DI → code khoa học, không spaghetti |
| **Frontend Framework** | **React 19 + Tailwind + shadcn/ui** | Component đẹp sẵn, Dark Mode, Accessibility |
| **Ngôn ngữ UI** | **100% Tiếng Việt** | Liva system prompt: GenZ/thân thiện tiếng Việt |
| **GPU (Optional)** | RTX 3060+ / VRAM ≥ 8GB | Ollama + Llama-3-8B offline chat miễn phí |

## 14. Bước Đầu Tiên: Viết Docker Compose

Bản thiết kế này đã **sẵn sàng để code ngay**. Bước đầu tiên cần làm:

1. **Tạo `docker-compose.yml`** hoàn chỉnh (đã có template ở Part 1)
2. **Tạo `.env`** từ `.env.example`
3. **`docker compose up -d`** → verify PostgreSQL + Redis + Nginx chạy
4. **`nest new server`** → scaffold NestJS project
5. **Viết Auth module** với Invite-Only registration

> 💡 **Tip**: Chạy `docker compose up -d postgres redis` trước để test DB connection, sau đó thêm dần các services khác.
